Book reliable rides for people & goods

Professional vehicle booking for transport. Calculate fare by distance, manage bookings, and coordinate partners — all from one page.

How fare is calculated

Base fare + (per‑km × distance) × surge. Admin can set these values in Settings.

  • Base fare (default ₹50)
  • Per‑km (default ₹12)
  • Optional surge (1.0 = no surge)

Distance options

Enter your own distance (km) or let us estimate using pickup/drop coordinates (Haversine).

  • Manual distance input
  • Coordinate-based estimate
